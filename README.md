## 💡 Features

<ul>
  <li>Holds and dries <b>6 umbrellas simultaneously</b></li>
  <li><b>Enclosed chamber</b> for safe and efficient drying</li>
  <li><b>3kW hot air blower</b> for fast drying</li>
  <li><b>Front-mounted 20x4 LCD</b> with buttons for controls</li>
  <li>Galvanized steel internal frame and airflow guide</li>
  <li>Galvanized steel exterior for durability</li>
</ul>

## 🧰 How to Use (Arduino UNO)

### 1. Clone or Download the Repository

```bash
git clone https://github.com/qppd/Umbrella-Holder.git
```

Or download ZIP from [this link](https://github.com/qppd/Umbrella-Holder/archive/refs/heads/main.zip), then extract it.

---

### 2. Open the Project in Arduino IDE

1. Launch the **Arduino IDE** (install from [arduino.cc](https://www.arduino.cc/en/software) if not yet installed).
2. Click **File > Open...**
3. Navigate to the folder:  
   `Umbrella-Holder/UmbrellaDryer.ino`
4. Select and open the `.ino` file.

---

### 3. Connect Your Arduino UNO

1. Plug in your **Arduino UNO** using a USB cable.
2. Go to **Tools > Board** and select **Arduino UNO**.
3. Go to **Tools > Port** and select the port where your board is connected.

---

### 4. Upload the Code

Click the **Upload** button (⭳) or press `Ctrl+U`.  
Wait for the message: **"Done uploading."**

---

### 5. Start Testing

- The LCD will display initial info.
- Use the buttons to interact with the drying system interface.
- Hot air system and servos should respond accordingly (if connected).
